# Structure

## Architecture

MUSUBI follows a modular architecture with the following components:

### Core Components
- `src/` - Source code (agents, analyzers, generators, validators)
- `bin/` - CLI entry points
- `steering/` - Project memory and governance
- `tests/` - Test suites (4,290+ tests)

### Key Modules
- `src/integrations/codegraph-mcp.js` - CodeGraph MCP v0.8.0 integration
- `src/agents/` - 27 specialized agents
- `src/performance/` - Enterprise performance optimization
- `src/enterprise/` - Multi-tenant and AI capabilities
