# Development Workflow

Common development workflows for musubi-sdd.

## Setup

```bash
# Installation
npm install

# Configuration
cp .env.example .env
```

## Development

```bash
# Start development server
npm run dev

# Run tests
npm test

# Lint code
npm run lint
```

## Build & Deploy

```bash
# Build for production
npm run build

# Deploy
npm run deploy
```
